# 《C++ Primer》笔记

## 1.标准库IO对象

- **cin:**标准输入，可以跳过空格、制表符、换行符等等
- **cout:**标准输出
- **cerr:**标准错误
- **clog:**用于输出程序运行时的一般信息

## 2.C++算数类型

**float:**6位有效数字

**double:**10位有效数字

**long double:**10位有效数字

```cpp
bool b=42;//非0，即真
int i=b;//真，值为1
```

不要混用带符号和无符号的变量，会转成unsigned变量

## 3.复合类型

- **引用：**为对象起的别名，引用本身不是对象，所以不能定义引用的引用。
- **指针：**指针是对地址的封装，本身就是一个对象。

**void*：**纯粹的地址封装，与类型无关。可以用于存放任意类型的地址
